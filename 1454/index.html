前言

在用VPS安装Python环境的时候有出现”-bash: make: command not found“提示问题，一般我们很多时候安装软件也会出现这样的问题。

解决方法

Centos

yum -y install gcc automake autoconf libtool make

Debian或者Ubuntu

apt-get install gcc automake autoconf libtool make
