一个非常简单的多线程下载工具，解决单线程下载速度慢的问题，首先是安装：

apt-get install axel #debian/ubuntu安装方法
yum install axel #centos安装方法

使用方法，就几个参数，非常简单：

axel -n 10 http://......


-o /tmp #指定下载目录
-q #静默模式
-s #限制最大速度
