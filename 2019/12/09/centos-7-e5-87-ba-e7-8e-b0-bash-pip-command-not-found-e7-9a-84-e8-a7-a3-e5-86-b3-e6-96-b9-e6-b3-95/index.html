说明：一般我们在centos 7上使用pip命令遇到-bash: pip: command not found的时候，通常会按照网上的教程，先yum安装epel再安装pip来解决，可是有时候却发现不行，会提示No package python-pip available，这里我说个通用的方法。

方法

运行命令即可安装pip：

wget https://bootstrap.pypa.io/get-pip.py
python get-pip.py
