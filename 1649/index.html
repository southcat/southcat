简介

一个基于html5画布绘制的网页背景效果，非常赞！如果需要wordpress插件，在插件库搜索canvas-nest或者看项目canvas-nest-for-wp。

在线演示：http://www.atool.org/。

特征

  • 不依赖于jQuery或其他JavaScript框架。
  • 很轻，只有1.6kb。
  • 所以很容易使用，即使你不是网页开发者。

用法

在<body>和</body>中间插入这段js代码即可，注意不要放在head里面。

< script  src = “ //cdn.bootcss.com/canvas-nest.js/1.0.1/canvas-nest.min.js ” > </ script >

如下所示：

< html >
< head >
    ...
</ head >
< body >
    ...
    ...
    ...
    < script  src = “ //cdn.bootcss.com/canvas-nest.js/1.0.1/canvas-nest.min.js ” > < / script >
</ body >
</ html >

配置

  • color：画布线颜色，默认值：’0,0,0’; 颜色为（R，G，B）。

  • opacity：线的不透明度（0〜1），默认值：0.5。

  • count：行数，默认值：150。

  • zIndex：z空间的索引，默认值：-1。
