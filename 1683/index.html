说明：我们很多时候都要用到ipmitool，这是说下方法。

首先需要启用两个内核模块：

modprobe ipmi_si
modprobe ipmi_devintf
echo ipmi_si >> /etc/modprobe
echo ipmi_devintf >> /etc/modprobe

然后安装ipmitool：

apt-get install ipmitool

列出IPMI网络信息：

ipmitool lan print


ipmitool user list 1
