说明：有时候换系统尝试运行Python程序时，服务器会返回-bash: python: command not found错误指令，说明系统上未安装Python，这里说下方法。

方法

1、安装
Ubuntu/Debian系统：

apt-get install python
或者
apt-get install python3

CentOS系统：

yum install python

2、检查

type -a python
which python
ls -l /usr/bin/python
ls -l /usr/bin/python*
